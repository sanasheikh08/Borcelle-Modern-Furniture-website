# Borcelle Modern Furniture Company

A modern furniture website showcasing indoor, outdoor, and office products with responsive design.

## Features

- Responsive homepage with hero section
- Product galleries for indoor, outdoor, and office furniture
- Navigation bar and interactive elements using JavaScript
